# Regression Analysis of Stock Returns

## Objective
To study the relationship between company financial performance and stock returns using regression analysis.

## Companies Studied
- Reliance Industries (Product-based)
- Dixon Technologies (Product-based)
- Tata Consultancy Services (Service-based)

## Methodology
- Data collected from Yahoo Finance and Screener
- Stock returns calculated using percentage change formula
- Multiple linear regression applied using EPS and P/E ratio

## Tools Used
- Microsoft Excel
- AI-assisted data structuring and regression
- GitHub for version control

## Disclaimer
This project is for academic purposes only and does not constitute investment advice.
